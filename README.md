## 📁 Repository 안내

- 이 저장소는 **SQL 문제 풀이 및 쿼리 연습 기록**을 정리한 공간입니다.  
- 코딩 테스트, 데이터 분석, 실무 대비를 위해 다양한 플랫폼의 SQL 문제를 풀이하고 기록합니다.
- SQL 문제 풀이 기록을 **플랫폼 및 업로드 방식에 따라 체계적으로 관리**합니다.

> 📌 **본 저장소에는 2026/01/13 이후에 풀이한 문제들만 기록되어 있으며, 그 이전에 풀이한 문제들은 포함되어 있지 않습니다.**


## ℹ️ 참고 사항

- 동일한 플랫폼이라도 업로드 시점에 따라 폴더 구조가 달라질 수 있습니다.
- 쿼리는 **정답 + 가독성 + 유지보수성**을 기준으로 지속적으로 개선합니다.
- DB 종류(MySQL, Oracle, PostgreSQL 등)에 따라 문법 차이가 일부 존재할 수 있습니다.
